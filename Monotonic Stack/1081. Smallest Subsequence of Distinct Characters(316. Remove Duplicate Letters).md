Given a string s, return the lexicographically smallest subsequence of s that contains all the distinct characters of s exactly once.

 

Example 1:

Input: s = "bcabc"
Output: "abc"
Example 2:

Input: s = "cbacdcbc"
Output: "acdb"
 

Constraints:

1 <= s.length <= 1000
s consists of lowercase English letters.

Thought:

In this question, we need to satisfy two requirements: 1. all distinct letters need to appear only once in string
2. The string needs to be lexicographically smallest.
We need to maintain a monotonic stack and array of last index where the character appears in the original string.
When we iterate the original string, if we find the current character is smaller than the top of stack and the last index
of the top of the stack would be appeared behind the current character, then we could remove the top of stack safely.
Then add the current character to the stack.

Solution:

```
class Solution {
    public String smallestSubsequence(String s) {
        int[] last = new int[26];
        boolean[] seen = new boolean[26];

        Stack<Character> stack = new Stack();
        int n = s.length();

        for (int i = 0; i < n; i++) last[s.charAt(i) - 'a'] = i;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            if (seen[c - 'a']) continue;

            while (!stack.isEmpty() && stack.peek() > c && last[stack.peek() - 'a'] > i) {
                seen[stack.pop() - 'a'] = false;
            }

            stack.push(c);
            seen[c - 'a'] = true;
        }

        StringBuilder sb = new StringBuilder();
        for (char c : stack) sb.append(c);
        return sb.toString();
    }
}
```
